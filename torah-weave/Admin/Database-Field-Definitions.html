Fields capturing literary and theological content

Field Name Type Format Example When to Update

literary_techniques Array Comma-separated chiasm, envelope, triadic After Part D written
visual_hierarchy Text Spatial description Center position emphasizes covenant After Part D written
supernatural_content Boolean TRUE/FALSE TRUE After Part D written
Update Trigger: Complete after Part D (Unit Commentary) finished.

Movement Types:
  • unity_to_separation - Things start together, end apart
  • separation_to_unity - Things start apart, come together
  • intensification - Pattern builds/escalates
  • circular - Returns to starting point
  • static - No movement, establishes baseline

Literary Techniques Vocabulary:
  • chiasm - ABBA or ABCBA pattern
  • envelope - Same material at beginning and end
  • triadic - Three-part structure
  • concentric_rings - Nested layers
  • parallel - Similar structures repeated
  • inversion - Pattern reversed

Data Entry Best Practices

Consistency Standards

1. Divine Names:

  • Always capitalize: YHWH, Elohim
  • Use compound form: YHWH Elohim (not "Yahweh-Elohim")

2. Verse References:

  • Full format: Genesis 11:10-13:4
  • No abbreviations in verse_range field

3. Boolean Fields:

  • Use Excel TRUE/FALSE (not yes/no or 1/0)

4. Array Fields:

  • Comma-separated with single space: GEN-05, GEN-12, GEN-16
  • No brackets or quotes

5. Empty Fields:

  • Leave blank (not "N/A" or "none")
  • Exception: subdivision_pattern should be "none" if no subdivisions

6. Thematic Descriptions:

  • Be specific but concise (aim for 5-15 words)
  • Focus on observable structural features, not interpretations

Common Issues and Solutions

Issue: Not sure about warp vs. weft themes

Solution:

  • Warp (columns/poles): Read DOWN each column - what's the thematic unity?
  • Weft (rows/threads): Read ACROSS each row - what's the thematic progression?

Issue: Unit has irregular structure

Solution:

  • Set structure_type: irregular
  • Set dimensions to actual pattern (e.g., 23332)
  • Note irregularity in notes field

Issue: Multiple movement types seem to apply

Solution:

  • Choose the DOMINANT pattern
  • Note secondary patterns in notes field

Issue: Uncertain about triad membership

Solution:

  • Set confidence_level: medium or low
  • Document uncertainty in notes field
  • Mark for review after more units analyzed

Database Maintenance

Monthly Tasks:

  • Backup database file with date stamp
  • Review analysis_status field for stalled units
  • Check for missing cross-references as new units completed

After Each Book Completed:

  • Verify all units have analysis_status: complete
  • Check that all triad relationships documented
  • Update thematic_chain fields as patterns across books emerge
