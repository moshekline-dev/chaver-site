Torah Unit Commentary Taxonomy Template

I. UNIT HEADER ELEMENTS

A. Visual Elements (Required)

  1. Iconic Book Map

    • SVG mini-map showing all units in the book
    • Current unit highlighted in contrasting color
    • Positioned at top of page for immediate context

  2. Full Unit Text

    • Complete text from woven site with verse references
    • Color-coded according to literary patterns
    • Presented in appropriate table structure (regular/irregular)
    • Color legend for pattern identification

B. Metadata (Required)

  1. Unit Title: [Book] Unit [#]: [Descriptive Name]
  2. Structure: [Dimensions] (e.g., 4×2 Regular, 13331 Irregular)
  3. Biblical Reference: [Book Chapter:Verse-Chapter:Verse]
  4. Theme: [Primary thematic focus]
  5. Position: Row [#] Column [Letter] in book matrix

C. Navigation (Required)

  1. Previous Unit link (if applicable)
  2. Next Unit link (if applicable)
  3. Return to Book Overview link
  4. Breadcrumb trail: Torah Map > [Book] Map > Unit [#]

II. INTRODUCTORY FRAMEWORK

A. Opening Hook (Required)

  • Compelling quote or observation that captures the unit's essence
  • Connection to contemporary relevance or timeless question
  • Sets tone for deeper exploration

B. Unit Context (Required)

  1. Position in Book Architecture

    • Location in the book's matrix (row/column significance)
    • Relationship to surrounding units (pairs, triads, sequences)
    • Role in larger book structure

  2. Relationship to Adjacent Units

    • If part of a diptych/triptych, explain the connection
    • Continuity or contrast with previous/next units
    • Shared vocabulary or thematic links

  3. Speech/Narrative Context (where applicable)

    • Which speech or narrative section contains this unit
    • Speaker identification and audience
    • Historical/literary setting

III. STRUCTURAL ANALYSIS

A. Dimensional Overview (Required)

  1. Structure Type

    • Regular (all rows same width) vs. Irregular (symmetrical variation)
    • Notation explanation (e.g., 4×2 = 4 rows, 2 columns each)
    • Special features (framework structure, independent unit, etc.)

  2. Organizational Principle

    • Primary organizing logic (temporal, spatial, thematic, etc.)
    • How the structure serves the content
    • Comparison to similar structures in Torah

B. Thread Analysis (Required)

  1. Warp Threads (Columns)

    • Identification and labeling of each column
    • Thematic or categorical principle for each
    • Progression or relationship between columns

  2. Weft Threads (Rows)

    • Identification and labeling of each row
    • Sequential or hierarchical principle
    • Movement or development across rows

C. Matrix Presentation (Required)

  • Visual matrix/table showing the unit's structure
  • Clear labeling of all intersections
  • Brief content summary in each cell
  • Pattern identification (parallels, chiasms, etc.)

IV. DETAILED COMMENTARY

A. Cell-by-Cell Analysis (Flexible depth based on unit)

  1. Intersection Readings

    • What each intersection reveals
    • Key vocabulary and phrases
    • Divine name usage (YHWH/Elohim patterns)

  2. Pattern Recognition

    • Horizontal parallels and their significance
    • Vertical connections and progressions
    • Diagonal or chiastic relationships
    • Closure/frame elements

  3. Subdivision Analysis (where applicable)

    • Function of subdivisions (A, B, C within cells)
    • Whether they create sub-rows or serve as parallel indicators
    • Micro-patterns within larger structure

B. Thematic Development (Required)

  1. Major Themes

    • Primary theological/philosophical concepts
    • How structure reinforces meaning
    • Evolution of themes through the unit

  2. Literary Techniques

    • Repetition, wordplay, inclusio
    • Narrative devices and rhetorical strategies
    • Use of formulaic language

  3. Intertextual Connections

    • Links to other Torah units
    • Connections to broader biblical literature
    • Echoes and allusions

V. THEOLOGICAL IMPLICATIONS

A. Divine-Human Dynamics (Required)

  1. Divine Name Analysis

    • Distribution of YHWH vs. Elohim
    • Correlation with content and structure
    • Theological significance of usage patterns

  2. Covenant Relationships

    • How unit portrays divine-human interaction
    • Individual vs. communal dimensions
    • Conditional vs. unconditional elements

B. Deeper Meanings (Required)

  1. Woven Reading Insights

    • Meanings visible only in 2D structure
    • Emergent themes from pattern recognition
    • Hidden unity revealed through weaving

  2. Transformative Elements

    • How the unit transforms understanding
    • Shifts in perspective or theology
    • Revolutionary vs. evolutionary ideas

VI. SYNTHESIS AND CONCLUSIONS

A. Unit Contribution (Required)

  1. To Book Structure

    • How this unit advances the book's argument
    • Essential vs. supplementary role
    • Structural necessity

  2. To Torah Theology

    • Unique theological contribution
    • Place in Torah's overall message
    • Universal vs. particular elements

B. Contemporary Relevance (Optional but encouraged)

  1. Timeless Principles

    • Extracting eternal truths from historical context
    • Application to modern situations
    • Pedagogical value

  2. Questions for Reflection

    • Open-ended questions for further study
    • Personal application prompts
    • Community discussion topics

C. Summary Statement (Required)

  • Concise encapsulation of unit's significance
  • Connection back to opening hook
  • Forward link to next unit (if applicable)

VII. SUPPLEMENTARY ELEMENTS

A. Technical Notes (As needed)

  1. Textual Variants

    • Significant manuscript differences
    • Translation challenges
    • Scholarly debates

  2. Historical Context

    • Archaeological insights
    • Ancient Near Eastern parallels
    • Historical-critical observations

B. Study Aids (Optional)

  1. Key Terms Glossary

    • Hebrew terms with transliteration
    • Technical vocabulary explained
    • Conceptual definitions

  2. Further Reading

    • Scholarly sources
    • Related commentaries
    • Cross-references to other project materials

VIII. FORMATTING STANDARDS

A. Length Guidelines

  • Minimum: 2,000 words (simple units)
  • Standard: 3,000-5,000 words (typical units)
  • Maximum: 8,000 words (complex central units)

B. Visual Elements

  • Required: Book map, unit text, structural matrix
  • Recommended: Comparison tables, pattern diagrams
  • Optional: Additional charts, word studies

C. Citation Style

  • Biblical verses: [Book Chapter:Verse]
  • Cross-references to other units: "See [Book] Unit [#]"
  • Scholarly sources: Footnotes or endnotes

IX. QUALITY CHECKLIST

Before finalizing any unit commentary, verify:

  • [ ] Iconic book map with unit highlighted
  • [ ] Full unit text with color coding
  • [ ] Clear structural analysis with matrix
  • [ ] Thread identification (warp and weft)
  • [ ] Pattern recognition and explanation
  • [ ] Divine name analysis
  • [ ] Theological implications explored
  • [ ] Connection to book architecture explained
  • [ ] Summary captures unit's contribution
  • [ ] Navigation elements functional
  • [ ] Accessible language (avoiding unnecessary jargon)
  • [ ] Consistent formatting throughout

X. ADAPTATION NOTES

This taxonomy should be adapted based on:

  1. Unit Complexity

    • Simple units may not need every subsection
    • Complex units may require additional analysis
    • Independent units need special treatment

  2. Book Characteristics

    • Genesis: Creation themes, universal to particular
    • Exodus: Liberation and law themes
    • Leviticus: Holiness and ritual themes
    • Numbers: Wilderness and organization themes
    • Deuteronomy: Covenant renewal and future vision

  3. Structural Type

    • Regular units: Focus on systematic patterns
    • Irregular units: Explain symmetrical variation
    • Independent units: Emphasize unique role

This taxonomy provides a comprehensive framework while allowing flexibility for each unit's unique characteristics.
